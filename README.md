# CourseTable2ICS

Convert course table to ICS for SSPU

## 快速开始

### 安装依赖

``` bash
$ pip install -r requirements.txt
# maybe pip3
```

### 运行

``` bash
$ python main.py
# maybe python3
```
根据提示即可生成 ICS 文件。
